# Olympics Medal Analysis using Pandas

This is a beginner data analysis project created using Python and Pandas.  
The project analyzes Olympics medal data to find insights such as top countries, athletes, sports, and medal counts.

## Tools Used
- Python
- Pandas
- Matplotlib

## Features
- Data cleaning
- Handling missing values
- Filtering and sorting data
- GroupBy analysis
- Medal count analysis
- Basic charts and visualization

## Project Goal
To practice Pandas and perform basic data analysis on a real-world dataset.
